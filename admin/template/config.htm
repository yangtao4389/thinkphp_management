调用方式:$howcalled
网站名称:网站名称，将显示在页面标题后边 webname
主页链接名:链接到首页时的名称 navname
站点访问url:例如:http://www.abc.com/末尾要加“/” 。 weburl
网站关键字:header 头中keyword部分 keyword
网站描述:header 头中description部分 description
网站统计代码:直接把html代码放进去 webtj
图片存储路径:例如 uploadpic/注意前边没有'/' uploadpicpath
网站主服务QQ:当前服务网站主的qq,当一个人数基本够时,可设置成另一个 weberserviceqq
广告主服务QQ:当前服务广告主的qq,当一个人数基本够时,可设置成另一个 aderserviceqq
文章列表页默认模板:不包含网站根目录,例如:article/template/default/list.htm deftmplist
文章内容页默认模板:不包含网站根目录,例如:article/template/default/content.htm deftmpcontent
图片访问域名:例如:http://img11.2366.com artimgdomain
水印图片路径:例如images/mark.png waterpath
